# Email Stack 

A simply base to develop a responsive email.  
It's a responsive template (width 600/300 pixels on desktop/mobile)

---

### Pattern

IMAGE
> x1 COLUMN  
> x1 COLUMN WRAPPER  
> x2 COLUMN (FIX)  
> x2 COLUMN (SWITCH)  
> x2 COLUMN WRAPPER (FIX)  
> x2 COLUMN WRAPPER (SWITCH)  
> x3 COLUMN (FIX)  
> x3 COLUMN (SWITCH)  
> x3 COLUMN WRAPPER (FIX)  
> x3 COLUMN WRAPPER (SWITCH)
